## About the Project
This Power BI project analyzes four years of store sales data for EmporiUm, a virtual student bookstore. The dashboard was created to help identify sales trends, compare regional performance, and evaluate product category performance. The report allows users to interact with the data through filters and visualizations to gain meaningful business insights.

## What I Did
- Imported and cleaned the dataset using Power Query.
- Built a data model by creating relationships between tables.
- Created DAX measures to calculate key sales metrics.
- Designed an interactive dashboard with charts, tables, KPI cards, and slicers.
- Analyzed sales by year, region, state, and product category.

## Key Insights
- Identified the highest-performing regions and states.
- Compared sales trends over multiple years.
- Evaluated which product categories generated the most sales.
- Created visuals that make it easy to explore sales performance.
